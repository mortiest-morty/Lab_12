Вариант 7

STL2Seq6

Даны вектор V, дек D и список L. Каждый исходный контейнер содержит не менее 
трех элементов, количество элементов является нечетным. Удвоить значения первого, 
среднего и последнего элемента каждого из исходных контейнеров.  

STL2Seq14

Дан вектор V. Вставить после каждого элемента исходного вектора число −1. 
Использовать функцию-член insert в цикле с параметром-итератором. 

STL2Seq22

Даны список L и вектор V; список L имеет нечетное количество элементов. 
Переместить средний элемент списка L в конец вектора V. Использовать функции-члены 
push_back и erase.

STL1Iter18

Дана строка name и целое число K (> 0). Записать в текстовый файл с именем 
name K символов «*». Использовать алгоритм fill_n.
